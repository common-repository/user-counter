== Description ==

User Counter allows you to display a total count of all users in your WordPress site in any post, page or widget.
See http://anthony.strangebutfunny.net/my-plugins/user-counter/ for help
 
== Installation ==

1. Upload `the directory "user-counter"` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "User Counter Settings" in the dashboard under Settings to configure it.
